Download Hungarian geocache data for printing

This website is a tool to download geocache descriptions and latest logs for printing from geocaching.hu (Note: this site is completely independent of geocaching.com; it has a different (but somewhat overlapping) set of caches and all the content is in Hungarian).

Comma separated list of cache IDs
A lot of cachers keep the default text of the logs when they submit them via apps; these logs are not very informative.
